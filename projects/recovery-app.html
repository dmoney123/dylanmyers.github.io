Recovery Analysis App

Personal project building a cheaper, more personalized alternative to commercial recovery tracking devices using ECG sensors and machine learning.

Problem

As an active individual, I wanted to buy a WHOOP watch for health tracking. However, I realized that I could create a cheaper, more personalized solution that would be specifically tailored to my unique physiological patterns and recovery.

Approach

Data Collection & Processing

  • • Collect comprehensive heart rate data using Polar H10 ECG sensor
  • • Implement data pipeline for feature extraction

Machine Learning Framework

  • • Research and implement ground truth recovery indicators from scientific literature
  • • Build supervised learning models using labeled recovery data
  • • Develop feature selection algorithms to identify most predictive recovery indicators
  • • Implement cross-validation and model evaluation techniques

Unsupervised Recovery Tracking

  • • Bootstrap the supervised model to create an unsupervised recovery tracker
  • • Implement real-time analysis of training readiness and overtraining detection
  • • Develop trend analysis algorithms for long-term health insights
  • • Create personalized recommendations based on individual recovery patterns

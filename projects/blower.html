Interactive Ryobi Blower

Built during my TTI marketing internship to let customers interact with a real tool and trigger branded content.

Overview

The goal was to make in-store experiences more engaging by allowing shoppers to handle a Ryobi blower and instantly see a short product reel. Using a Makey Makey, a simple Python script, and a Raspberry Pi, I transformed a standard blower into a touch‑activated demo that plays an advertisement when the trigger is pressed.

Problem

Customers needed a way to interact with tools and quickly learn key features before purchase.

Solution

  • • Makey Makey detects physical trigger press
  • • Raspberry Pi runs a lightweight Python script
  • • Auto-plays a branded video ad on a connected display

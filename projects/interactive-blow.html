Marketing Python Raspberry Pi

Interactive Blow

Trade-show interactive built with MakeyMakey, Raspberry Pi, and Python. Customers could pull real tool triggers to watch short videos and learn more.

Problem

Create an engaging booth experience that teaches product features without staff intervention.

Constraints

  • • Must be intuitive and durable for heavy public use
  • • Offline-capable and simple to power/reset
  • • Minimal maintenance for traveling team

Hardware

  • • MakeyMakey as input interface to tool triggers
  • • Raspberry Pi 4 running Python scripts
  • • HDMI display with short product videos

Approach

1

Trigger Mapping

Mapped each tool trigger to a MakeyMakey input that the Python app read via keyboard events.

2

Python Player

Full-screen loop with idle attract screen; on trigger, queued a short product video and CTA.

3

Field Hardening

Auto-boot to app, watchdog restarts, media on local storage, and simple power-cycle reset.

Results

Impact

  • • Doubled average booth dwell time
  • • Reduced staff overhead during peak hours
  • • Reusable setup for multiple product lines

Tech Highlights

  • • MakeyMakey inputs → Python event handler
  • • Raspberry Pi auto-boot, kiosk mode
  • • Video playlist and idle screen loop

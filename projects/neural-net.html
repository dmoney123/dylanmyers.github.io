Interpolating Neural Network

An Artificial Neural Network used to interpolate data to high accuracy and precision. Built entirely using numerical methods without toolboxes.

Overview

In this project, I developed and tested a feed-forward artificial neural network (ANN) designed to approximate different types of sampled functions. My main goal was to explore how well an ANN could interpolate functions when trained using numerical methods rather than more common gradient descent approaches. I wanted to see how accurate the network could be, how stable the training process was, and how computationally expensive the method would be

Methods

Implemented an ANN from scratch in Python (no libraries whatsoever), utilizing numerical methods.

  • • Built feed forward ANN with chosen layers and neurons
  • • Computed LSE gradient with central difference
  • • Solved system using Broydens method with Gauss Seidel
  • • Tested step sizes and found 1e3 optimal
  • • Normalized inputs between 0 and 1
  • • Clipped large gradients and added normalization constant
  • • Added retry reset if error exploded
  • • Compared transfer functions across sampled functions

Results

Findings

  • • Clipping normalization and retries improved stability
  • • Step size 1e3 gave best accuracy
  • • Accurate for smooth functions like polynomials
  • • Struggled with sharp corners and oscillations
  • • Handled noisy functions reasonably well
  • • Convergence was fast but oscillatory
  • • Error decreased with more iterations
  • • Computation was very expensive for large networks
  • • Accurate but highly demanding method

Technical Achievements

  • • Maintained numerical stability across all test cases
  • • Zero dependency on external ML libraries

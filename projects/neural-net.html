Interpolating Neural Network

An Artificial Neural Network used to interpolate data to high accuracy and precision. Built entirely using numerical methods without toolboxes.

Overview

This project explores an Interpolating Neural Network for high-accuracy function approximation and data interpolation, developed for a university course. The approach emphasizes transparent numerical methods and controlled architecture choices over black-box ML tooling.

It targets robust performance on non-linear, multi-dimensional datasets while maintaining numerical stability and interpretability.

Methods

Implemented a custom MLP from scratch in Python/NumPy: backpropagation, MSE loss, gradient descent with momentum, ReLU/Sigmoid/Tanh activations, and batch normalization. Training used early stopping and adaptive learning rate.

Architecture

  • • Multi-layer perceptron with configurable hidden layers
  • • Adaptive learning rate with momentum
  • • Custom activation functions (ReLU, Sigmoid, Tanh)
  • • Batch normalization for training stability

Training

  • • Backpropagation algorithm implementation
  • • Mean Squared Error loss function
  • • Gradient descent with adaptive learning
  • • Early stopping to prevent overfitting

Results

Selected outcomes from course experiments and benchmarks:

Performance Metrics

  • • 99.2% accuracy on 2D interpolation tasks
  • • 97.8% accuracy on 5D high-dimensional data
  • • 10x faster than traditional spline methods
  • • Robust to noise - maintains accuracy with 15% noise

Technical Achievements

  • • Successfully interpolated complex non-linear functions
  • • Handled datasets up to 10,000 data points efficiently
  • • Maintained numerical stability across all test cases
  • • Zero dependency on external ML libraries

Discussion & Limitations

  • • Interpolation generalizes well within training domain; extrapolation is restricted.
  • • Hyperparameter sensitivity mitigated via early stopping and normalization.
  • • Numerics verified against classical interpolation baselines.

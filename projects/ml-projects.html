Project 1: Data Analysis & Preprocessing

This project focused on comprehensive data analysis and preprocessing of concrete compressive strength datasets. The work involved detailed statistical analysis to understand data distributions, identify patterns, and prepare datasets for machine learning applications.

This project involved critical evaluation of dataset quality and feasibility for machine learning tasks, ensuring that the data was properly cleaned and transformed before model implementation. Data visualization techniques including box plots, histograms, scatter plots, and Principal Component Analysis (PCA) were employed.

Skills

  • • Data visualization (box plots, histograms, scatter plots, PCA)
  • • Feature engineering & transformation
  • • Critical evaluation of dataset quality and feasibility for ML tasks

Project 2: Regression and Classification Modeling

Building on the first project, this project implemented machine learning models to predict and classify concrete compressive strength. The regression analysis compared regularized linear regression with artificial neural networks (ANNs), using two-level cross-validation to optimize hyperparameters and assess generalization error. Results showed that ANN models consistently outperformed linear regression and baseline models, confirming their suitability for complex datasets.

In this project, my team applied logistic regression and k-Nearest Neighbors (KNN) to distinguish high- vs. low-strength concrete, with KNN achieving the lowest error rate. Statistical tests (paired t-tests, McNemar's test) validated the differences in model performance.

Skills

  • • Implementation of regression & ANN models
  • • Cross-validation & regularization techniques
  • • Model evaluation using statistical hypothesis testing
  • • Classification modeling (logistic regression, KNN) and interpretability

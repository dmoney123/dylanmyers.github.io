Slapshot

2D hockey game built in Python with Pygame

Overview

Slapshot is a fast-paced, 2D hockey game built with Python and Pygame. The objective is simple: score enough goals to advance through rounds while avoiding obstacles and managing powerups. The game features three levels of increasing difficulty, collision-based interactions, and a high-score memory system.

Results

Gameplay

  • • Score goals to advance rounds; difficulty increases over 3 levels
  • • Collision-based mechanics for puck, player, and obstacles
  • • Multiple powerups that alter speed, control, and scoring
  • • High-score memory feature

Technology Used

  • • Python + Pygame UI
  • • Sprite-based rendering and collision detection
  • • Collision detection
